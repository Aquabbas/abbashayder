## What are Component Islands?

- Render HTML on a web page using [[Server-Side Rendering (SSR)]]. Add
  placeholders/slots (مكان مؤقت) in more dynamic areas of the web page and
  output the server-rendered HTML there.

- Update these placeholders on the client-side, using [[Client-Side Hydration]].
  This is done on small [[Web Widgets]] (قطعة) of the web page.
