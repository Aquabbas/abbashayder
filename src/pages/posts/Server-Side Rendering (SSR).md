## What is Server-Side Rendering (SSR)?

- The ability to generate and send rendered HTML from the server to the client's
  web browsers. This approach uses less JavaScript, as there is no markup that
  has to be rendered into HTML.

- Benefits:
  - Search Engines can crawl and index the content, prior to delivery to the
    client, which enhances Search Engine Optimization (SEO).
  - Initial page loads faster, compared to Client-Side Rendering (CSR).
  - Ideal for Static Sites and Smaller Web Applications (Not ideal for Social
    Media Platforms where there is a lot of interactivity)
